# xeuclid
Extended Euclidean Algorithm based on `openssl`

## Requirement
- openssl/bn.h

## Syntax
```
./xeuclid <num a> <num b>
```

## Example
```
./xeuclid 123123123111 1293109238019381121
a =  123123123111
b =  1293109238019381121
x =  310080195628043417
y =  -29524220366
123123123111*(310080195628043417) + 1293109238019381121*(-29524220366) = 1

```
